第二节：数据的分类——数据类型 (Data Types)
程序离不开“数据”，而数据类型决定了：
- 这一块内存能存多大的数、多精确的数；
- 这些数可以参与哪些运算、运算时如何处理。
这一节先聚焦最常用的基本类型，以及一个非常重要的字符串类型。

---
2.1 常见基本数据类型概览——“几种常用数据，各司其职”
常见的基本数据类型可以粗略分为：
- 整数型：short, int, long, long long
- 浮点型：float, double
- 字符型：char
- 布尔型：bool
说明：
 不同平台上，每种类型占用的字节数可能不同。
 可以用 sizeof(类型) 或 sizeof(变量) 在当前环境中实际测试。

#include <iostream>
using namespace std;

int main() {

    cout << "sizeof(short)     = " << sizeof(short)     << " 字节\n";
    cout << "sizeof(int)       = " << sizeof(int)       << " 字节\n";
    cout << "sizeof(long)      = " << sizeof(long)      << " 字节\n";
    cout << "sizeof(long long) = " << sizeof(long long) << " 字节\n";
    cout << "sizeof(float)     = " << sizeof(float)     << " 字节\n";
    cout << "sizeof(double)    = " << sizeof(double)    << " 字节\n";

    return 0;
}

2.2 字符串类型 std::string——存“一整句文字”的类型
在 C++ 中：
- 单个字符 用 char 表示，使用单引号 ' '；
- 一串文本 用 std::string 表示，使用双引号 " "。
使用 std::string 需要：
#include <string>   // 头文件
using std::string; // 或者 using namespace std;
2.2.1 char 与 string 的基本使用
#include <iostream>
#include <string>

using namespace std;

int main() {
    // 字符类型：存一个字符，用单引号
    char letter = 'A';
    
    // 字符串类型：存一整串文字，用双引号
    string greeting = "Hello C++";
    string school   = "嘉庚学院";

    cout << "letter: " << letter << endl;
    cout << greeting << ", " << school << endl;

    return 0;
}
要点：
- char：'A', 'b', '0'
- string："Hello", "C++", "嘉庚学院"
- 必须包含头文件 #include <string>。

---
2.2.2 字符串的简单拼接与输出
最常见的两种用法：
1. 用 + 把多个字符串拼接成一个：
string name = "张三";
string hello = "你好，";
string end = "，欢迎学习 C++！";

string msg = hello + name + end;
cout << msg << endl;
1. 在 cout 中用 << 依次输出多个部分：
string school = "嘉庚学院";
string major  = "计算机科学";

cout << "我在" << school << "学习" << major << "。" << endl;

2.2.3 标准输出的“格式化”——控制显示的宽度和小数位
std::cout 是 C++ 中最常用的输出方式，可以结合 <iomanip> 头文件里的“控制符”对输出进行格式化。
常用控制符（都在 <iomanip> 中）：
- std::setw(n)：设置输出宽度为 n（只影响紧跟着的那一次输出）
- std::setfill(ch)：用指定字符 ch 来填充多出来的宽度
- std::setprecision(n)：设置浮点数输出的有效位数或小数位数
- std::fixed：配合 setprecision，改为固定小数位的表示方法
- std::scientific：用科学计数法输出浮点数
- 操纵符 std::left / std::right：控制左右对齐（结合 setw 使用）
<iomanip> 可以理解为：IO（输入输出） + manipulators（操纵符） 的合体。
示例 2.2-3：控制浮点数输出格式
#include <iostream>
#include <iomanip>  // 注意：使用 setw、setprecision 等需要这个头文件
using namespace std;

int main() {
    double a = 3.1415926;

    // 默认输出
    cout << "默认:       " << a << endl;

    // 只用 setprecision：控制有效数字个数
    cout << "有效数字(3): " << setprecision(3) << a << endl;

    // 使用 fixed + setprecision：控制小数位数
    cout << fixed; // 改为固定小数形式
    cout << "小数位(3):  " << setprecision(3) << a << endl;
    cout << "小数位(5):  " << setprecision(5) << a << endl;

    // 使用 scientific：科学计数法
    cout << scientific << setprecision(2);
    cout << "科学计数法: " << a << endl;

    return 0;
}
可以观察不同控制符对同一个 double 值输出效果的影响。
示例 2.2-4：控制输出宽度、对齐方式和填充字符
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int b = 255;

    // 默认输出
    cout << "默认:     " << b << endl;

    // 设置宽度为 5（不足的地方默认用空格在左边补齐）
    cout << "setw(5):  " << setw(5) << b << endl;

    // 设置宽度为 5，左侧用 '0' 补齐
    cout << "0填充:    " << setw(5) << setfill('0') << b << endl;

    // 恢复填充字符为空格
    cout << setfill(' ');

    // 设置宽度为 5，右对齐（这是默认）
    cout << "右对齐:   " << setw(5) << right << b << endl;

    // 设置宽度为 5，左对齐
    cout << "左对齐:   " << setw(5) << left  << b << endl;

    return 0;
}
输出示意（具体空格数可在编译器中实际查看）：
- 默认:     255
- setw(5):    255（左边有空格）
- 0填充:   00255
- 右对齐:    255
- 左对齐:   255  （右边有空格）
要点：
- setw(n) 只对紧跟着的那一次输出有效；
- setfill(ch) 会保留设置，直到再次修改；
- left / right 控制的是：内容在这块宽度中的对齐方式。

2.3 给类型起别名——“给复杂类型取个小名”
有些类型名字很长，例如：
- unsigned long long
- long long int
可以使用类型别名给它们起更短、更好记的小名。
2.3.1 typedef：传统写法
语法：typedef 原类型 别名;
typedef double Area;            // Area 代表 double
typedef unsigned long long ULL; // ULL 代表 unsigned long long
使用示例：
#include <iostream>
using namespace std;

typedef double Area;
typedef unsigned long long ULL;

int main() {
    Area circleArea = 3.14 * 2 * 2; // 相当于 double circleArea = ...
    ULL bigNumber = 10000000000ULL;

    cout << "circleArea = " << circleArea << endl;
    cout << "bigNumber  = " << bigNumber  << endl;

    return 0;
}
2.3.2 using：现代写法（C++11 推荐）
语法更直观：using 别名 = 原类型;
using Area = double;
using ULL  = unsigned long long;
使用方式相同：
#include <iostream>
using namespace std;

using Area = double;
using ULL  = unsigned long long;

int main() {
    Area roomArea = 20.5;     // 相当于 double roomArea = 20.5;
    ULL  count    = 123456789ULL;

    cout << "roomArea = " << roomArea << endl;
    cout << "count    = " << count    << endl;

    return 0;
}
要点：
- using 别名 = 原类型; 读起来更自然；
- 在现代 C++ 代码中，通常更推荐使用 using 来定义别名。
