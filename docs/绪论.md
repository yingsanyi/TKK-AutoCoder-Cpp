一、 为什么我们要学 C++？
你可能听说过 Python 简单好用，也听说过 Java 流行度高，但为什么在自动化专业，C++ 是不可撼动的基石？
1.1 自动化的“神兵利器”
想象一下，你在控制一个机械臂或无人机：
- 不仅要快： 自动化系统处理复杂的信号和控制任务，需要极高的运行效率。C++ 是编译型语言，速度极快。
- 还要稳： 你需要直接操作硬件接口（内存、指针）。C++ 允许你进行底层的“微操”，这是其他高级语言难以做到的 。
比喻： 如果说 Python 是全自动的“预制菜”，C++ 就是给你一把精密的“手术刀”，让你精准地控制每一个细节。
1.2 广阔的“钱”景
学会 C++，你的职业道路非常宽广：
- 嵌入式/硬件开发： 无人机、智能摄像头、医疗设备（薪资 15k-50k）。
- 游戏开发： 像腾讯、网易的游戏后台，C++ 是高性能游戏的首选。
- 金融科技： 高频交易系统，毫秒级的速度优势至关重要。

二、 什么是程序设计语言？
计算机只认识 0 和 1（二进制），我们人类说的是自然语言。程序设计语言就是这两者之间的翻译官 。
2.1 语言的进化阶梯 
1. 机器语言： 全是 010101，计算机直接读，但人类看天书。
2. 汇编语言： 用符号（如 ADD）代替 01，稍微好懂一点，但还是很繁琐。
3. 高级语言： 接近人类语言（如 x = x + y），C++ 就属于这一类。
2.2 翻译的两种方式 
高级语言计算机看不懂，需要“翻译”。
- 解释型（如 Python）： 像是同声传译。读一句，翻一句，执行一句。灵活但稍慢。
- 编译型（如 C++）： 像是笔译。把整本书（源程序）一次性翻译成机器码（可执行程序）。执行时不需要再翻译，所以速度飞快。

三、 C++ 程序是如何诞生的？
写好代码只是第一步，要让它跑起来，需要经历以下过程：
3.1 开发四部曲 
1. 编写源代码 (.cpp)： 你写下的如 Hello World 的代码，人类可读 (11)(11)(11)(11)。
2. 编译 (Compile)： 编译器检查语法，把代码翻译成目标代码 (.obj 或 .o) (12)(12)(12)(12)。
3. 链接 (Link)： 把你的目标代码和系统库文件“拼装”在一起。
4. 生成可执行文件 (.exe)： 最终生成的可以直接运行的程序 (13)。
3.2 程序的灵魂公式
计算机科学家 Niklaus Wirth 提出了一个著名公式：
$$程序 = 算法 + 数据结构$$
- 数据结构： 像是做菜的食材（数据的类型和组织形式）。
- 算法： 像是做菜的菜谱（操作步骤和流程）。
3.3 算法的三种基本逻辑 
无论多复杂的程序，都逃不过这三种结构：
1. 顺序结构： 一条路走到黑，按步骤 A -> B 执行。
2. 选择结构（分支）： 遇到路口，满足条件走左边，不满足走右边。
3. 循环结构： 只要条件满足，就在原地转圈圈（重复执行某段代码）。
4. (注意：循环必须要有出口，否则就是死循环！)

四、 动手实践：我的第一个 C++ 程序
4.1 认识开发环境 (IDE)
我们需要一个写代码和编译代码的软件。
- Visual Studio 2022： 微软出品，功能强大，是本次课程推荐的专业工具。
- Dev C++： 小巧简单，适合初学者快速上手。
4.2 Hello World 代码详解
这是每个程序员的“成人礼”：
#include <iostream>      // 1. 预处理：我要用输入输出工具箱
using namespace std;     // 2. 命名空间：使用标准工具箱的名字

int main()               // 3. 主函数：程序的入口，一切从这里开始
{
    cout << "Hello World!" << endl; // 4. 输出：在屏幕上打印文字并换行
    return 0;            // 5. 结束：返回0代表程序顺利跑完了
}
- cout：标准输出（屏幕），配合 << 像个喇叭一样把东西送出去 。
- endl：End Line，换行
- 注释：// 后面的内容是写给人看的笔记，计算机会忽略它。
4.3 变量与输入
程序不能只输出，还得能存数据、收数据。
- 变量：就像一个贴了标签的盒子，用来存数据。
  - int a; (声明一个叫 a 的整数盒子)
  - a = 5; (把 5 放进盒子)
  - string name = "嘉庚"; (存文本要用 string) (25)
- cin：标准输入（键盘），配合 >> 把键盘敲入的数据送到变量里 (26)(26)(26)(26)。
示例：加法计算器
int a, b;
cout << "请输入a的值：";
cin >> a;  // 等待用户输入a
cout << "请输入b的值：";
cin >> b;  // 等待用户输入b
cout << "a + b = " << a + b << endl; // 输出结果
