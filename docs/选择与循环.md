C++ 核心逻辑：掌握程序的“思考”与“耐力”
在程序设计的基础中，代码并不是永远只是一条直线走到底（顺序结构）。为了解决复杂问题，程序需要具备两个能力：
1. 判断能力（选择结构）： 像人一样，遇到分岔路口知道往哪走。
2. 坚持能力（循环结构）： 重复做同一件事，直到任务完成 (1)。

---
第一部分：教会程序“判断是非” (关系与逻辑运算)
在做决定之前，程序首先要学会比较。
1. 关系运算：比大小
这就像数学里的大小比较，用来告诉程序两个东西的关系。
- 符号： > (大于), < (小于), == (等于), != (不等于) (2)。
- 结果： 只有两种结果，要么是真 (true/1)，要么是假 (false/0) (3)。
- ⚠️ 致命陷阱： 在C++中，= 是赋值（给变量东西），== 才是判断相等。写错一个符号，程序逻辑就全乱了 (4)。
2. 逻辑运算：组合拳
有时候判断条件不止一个。比如：“只要你是男生 或者 你的分数大于90分”。
- && (逻辑与/AND)： 必须全对才是对。
  - 例子： 只有“有电” 并且 “开关打开”，灯才会亮 (5)。
- || (逻辑或/OR)： 只要有一个对就是对。
  - 例子： 只要“带了钥匙” 或者 “家里有人”，都能进家门 (6)。
- ! (逻辑非/NOT)： 唱反调。真变假，假变真 (7)。
💡 聪明的小技巧（短路运算）：
C++ 很懒。如果是 A && B，如果 A 已经是假了，它绝不会去算 B，因为结果注定是假。这叫“短路”。
例如： (i > 3) && (i++)，如果 i 是3，3>3 为假，后面的 i++ 根本不会执行，i 还是 3 (8)。

---
第二部分：选择结构 —— 程序的“大脑”
PPT中用扫地机器人的例子生动地解释了三种选择结构。
1. 单分支：直肠子 (if)
机器人一直走，只判断一种情况。
- 场景： 机器人检测前方有障碍物吗？
- 逻辑： 如果有（True），就停止。如果没有，什么都不做，继续原来的状态 (9)。
2. 双分支：二选一 (if-else)
机器人变聪明了，有了备选方案。
- 场景： 机器人检测前方障碍物。
- 逻辑： 如果有障碍物，就左转；否则（else），就前进 (10)。
- 代码结构：
if (前方有障碍) {
    左转;
} else {
    前进;
}
3. 多分支：层层筛选 (if-else if-else)
处理更复杂的情况，比如成绩评定。
- 场景： 根据分数给等级。
- 逻辑：
  1. 是不是 > 100 或 < 0？ -> 报错“不合法”。
  2. 如果不合法排除后，是不是 >= 95？ -> 优秀。
  3. 不够95，那是不是 >= 85？ -> 良好。
  4. ...以此类推。
- 注意： 这像下楼梯，一层层判断，满足了一层就不会去下一层了 (12)。
4. Switch 语句：对号入座
当你的选择是明确的“选项”（如星期一、星期二...）时，用 switch 更清晰。
- 场景： 输入1-7，输出星期几。
- 特点： 它像一个多路开关，直接跳到匹配的 case。
- ⚠️ 记得踩刹车： 每个 case 后面通常要加 break，否则它会一直往下执行（穿透） (13)。

---
第三部分：循环结构 —— 程序的“耐力”
当我们要输出 1 到 5，或者计算 1 加到 100，不可能写 100 行代码。这就需要循环。
1. while 循环：谨慎的跑者
- 口诀： 先判断，后执行。
- 逻辑： 只要条件满足（比如 i <= 5），就一直做。如果一开始条件就不满足，它一次都不做 (14)。
while (i <= 5) {
    跑圈;
    i++; // 别忘了计数，否则就死循环累死了
}
2. do-while 循环：冲动的跑者
- 口诀： 先执行，后判断。
- 逻辑： 不管三七二十一，先跑一圈，然后再看满不满足条件。
- 区别： 保证至少执行一次循环体 。
3. for 循环：专业的运动员
这是最常用的循环，结构最严谨。
- 格式： for (起跑线; 终点线; 步频)
- 例子： 累加求和
// i=1是初始化; i<=4是条件; i++是更新
for (int i = 1; i <= 4; i++) {
    sum += i;
}
4. 循环的控制：刹车与跳过
在循环中，我们可以随时干预：
- break (紧急刹车)： 直接跳出整个循环，不玩了。
- continue (跳过本轮)： 这一圈剩下的不跑了，直接回到起点开始下一圈 (17)。

---
第四部分：进阶 —— 循环嵌套
循环嵌套就是“圈里套圈”。最经典的例子是打印图形或乘法表。
例子：打印星星矩阵
如果你想打印一个 5行 x 5列 的星星方阵：
- 外层循环： 控制行数（管你有几行）。
- 内层循环： 控制每一行打印几个星星。
for (int i = 1; i <= 5; i++) {      // 外层：第1行到第5行
    for (int j = 1; j <= 5; j++) {  // 内层：每行打5个星
        cout << "*";
    }
    cout << "\n"; // 每行打完要换行
}
同理，打印九九乘法表或三角形，只需要修改内层循环的条件（比如 j <= i，让星星数量随行数增加） (19)。

---
总结
通过这节课，我们学会了如何指挥计算机：
1. 用关系和逻辑运算符 去看清世界。
2. 用 if 和 switch 去根据情况做决定（像扫地机器人避障）。
3. 用 while 和 for 去不知疲倦地重复工作。
